# ERC-8004 Identity

> Standard for on-chain AI agent identity — discoverable, verifiable, trustless.

## Overview

Agent Haus implements **ERC-8004** (ERC-8004: AI Agent Registry) to provide every agent with on-chain identity. This makes agents:
- **Discoverable** via agent explorers (8004scan, Agentscan)
- **Verifiable** with public metadata
- **Trustless** via smart contract enforcement

## What is ERC-8004?

ERC-8004 defines two registries:
- **Identity Registry** — Agent metadata (name, services, wallet)
- **Reputation Registry** — Feedback and reputation scores

## Registration Flow

```typescript
// 1. Prepare metadata
const metadata = {
  name: "Genesis",
  description: "AI trading agent on Celo",
  image: "ipfs://Qm...",
  agentWallet: "eip155:42220:0x...",
  services: [
    { name: "agenthaus-chat", endpoint: "https://agenthaus.space/api/chat" }
  ],
  registrations: [
    { agentRegistry: "eip155:42220:0x8004A169...", agentId: "2113" }
  ],
  supportedTrust: ["reputation"]
};

// 2. Upload to IPFS
const uri = await pinata.pinJSON(metadata);

// 3. Register on-chain
const tx = await identityRegistry.register(uri);
const agentId = await tx.wait(); // Token ID
```
